Style Guide

A quick overview of the typography, colors, and design language used throughout the design system.

Typography

Nayzak SAAS Template is following Bootstrap 5 basic typography system with some changes to match our template as well as additional font sizes for more flexibility.

Syne

Syne is a 5-style type family originally designed in 2017 for the art center Synesthésie, based in Saint-Denis in the very close suburb of Paris, France. Used across their identity, print and screen, Syne has been released under open-source license in June 2018.

Heading 1 Heading 2 Heading 3 Heading 4 Heading 5 Heading 6

Inter

Inter started out in late 2016 as an experiment to build a perfectly pixel-fitting font at (11px.) The idea was that by crafting a font in a particular way, it would be possible to get the best of both sharpness and readability.
